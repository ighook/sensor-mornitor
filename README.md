# Sensor Monitor

Sensor Monitor is a web application that allows users to monitor and manage their sensors.

## Features

- Real-time sensor data visualization
- Sensor data history
- Sensor data alerts
- Sensor data export

## Installation

```bash
npm install
```

## Usage

```bash
npm run dev
```
